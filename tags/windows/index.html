  • 方法1.

    或者添加到HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run下

    将一个.lnk文件添加到C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup目录下
